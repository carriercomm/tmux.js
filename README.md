# tmux.js #

A tmux plugin for [zsh.js](https://github.com/tadeuzagallo/zsh.js)


## Usage ##

```js
var zsh = require('zsh.js'),
    tmux = require('tmux.js');

zsh.create('container');
tmux.init(Zsh);
```
